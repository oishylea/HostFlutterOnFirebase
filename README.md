# Host Flutter on Firebase
Guide how to host a Flutter project on Firebase

## Step 1 : Create a project on Firebase
  <img src="HostFlutterOnFirebase/output/firebaseproject.png" alt="Firebase Main Page" style="height: 450px; width: auto; margin-right: 10px;" />

## Step 2 : Open the file on the command prompt
  <img src="HostFlutterOnFirebase/output/cpOpenFile.png" alt="Open File" style="height: 450px; width: auto; margin-right: 10px;" />

## Step 3 : Install Firebase
  <img src="HostFlutterOnFirebase/output/npminstall.png" alt="Install" style="height: 450px; width: auto; margin-right: 10px;" />
  <img src="HostFlutterOnFirebase/output/cpnpminstall.png" alt="Install on CP" style="height: 450px; width: auto; margin-right: 10px;" />


## Step 4 : Login Firebase
Type 'firebase login'
  <img src="HostFlutterOnFirebase/output/cplogin.png" alt="Install on CP" style="height: 450px; width: auto; margin-right: 10px;" />

## Step 5 : Initialize Firebase
Type 'firebase init'
  <img src="HostFlutterOnFirebase/output/cpinit.png" alt="Initialize Firebase" style="height: 450px; width: auto; margin-right: 10px;" />

## Step 6 : Build a web file for Flutter
Select 'n', then type 'flutter build web' <br>
  <img src="HostFlutterOnFirebase/output/cpbuildweb.png" alt="Build Web File" style="height: 450px; width: auto; margin-right: 10px;" /> <br>
After that, retype 'firebase init'

## Step 7 : Select Firebase feature
Enter the spacebar to select
  <img src="HostFlutterOnFirebase/output/cpfbfeature.png" alt="Select feature" style="height: 450px; width: auto; margin-right: 10px;" /> 

## Step 8 : Complete Firebase initialization
  <img src="HostFlutterOnFirebase/output/cpfbinit.png" alt="Firebase init" style="height: 450px; width: auto; margin-right: 10px;" /> 

## Step 9 : Complete Hosting
Type 'firebase deploy' to deploy the web file <br>
  <img src="HostFlutterOnFirebase/output/fbdeploy.png" alt="Deploy firebase" style="height: 450px; width: auto; margin-right: 10px;" />

  <img src="HostFlutterOnFirebase/output/hostlink.png" alt="Deploy firebase" style="height: 450px; width: auto; margin-right: 10px;" />
