
# Host Flutter on Firebase
Guide how to host a Flutter project on Firebase<br>

## Step 1 : Create a project on Firebase<br>
  <p align="center"><img src="output/firebaseproject.png" alt="Firebase Main Page" width="70%" /></p><br><br>

## Step 2 : Open the file on the command prompt<br>
  <p align="center"><img src="output/cpOpenFile.png" alt="Open File" width="70%" /></p>

## Step 3 : Install Firebase<br>
  <p align="center"><img src="output/npminstall.png" alt="Install" width="70%" /></p>
  <p align="center"><img src="output/cpnpminstall.png" alt="Install on CP" width="70%" /></p>


## Step 4 : Login Firebase<br>
Type 'firebase login' <br>
  <p align="center"><img src="output/cplogin.png" alt="Install on CP" width="70%" /></p><br><br>

## Step 5 : Initialize Firebase<br>
Type 'firebase init' <br>
  <p align="center"><img src="output/cpinit.png" alt="Initialize Firebase" width="70%" /></p><br><br>

## Step 6 : Build a web file for Flutter<br>
Select 'n', then type 'flutter build web' <br>
  <p align="center"><img src="output/cpbuildweb.png" alt="Build Web File" width="70%" /></p> <br>
After that, retype 'firebase init'<br><br>

## Step 7 : Select Firebase feature<br>
Enter the spacebar to select <br>
  <p align="center"><img src="output/cpfbfeature.png" alt="Select feature" width="70%" /></p> <br><br>

## Step 8 : Complete Firebase initialization<br>
  <p align="center"><img src="output/cpfbinit.png" alt="Firebase init" width="70%" /></p> <br><br>

## Step 9 : Complete Hosting<br>
Type 'firebase deploy' to deploy the web file <br>
  <p align="center"><img src="output/fbdeploy.png" alt="Deploy firebase" width="70%" /></p> <br><br>
